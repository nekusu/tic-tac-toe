# Tic Tac Toe

Tic Tac Toe game for [The Odin Project](https://www.theodinproject.com/).

## Build with

* HTML5
* CSS
* JavaScript

## Features

* Beautiful UI and animations.
* Edit your nickname.
* Play against an AI.
* AI vs AI!

You can play the finished game [here](https://nekusu.github.io/tic-tac-toe/).
